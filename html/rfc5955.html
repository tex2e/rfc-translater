1. Introduction
1. はじめに

This document defines a new media type for TimeStampedData envelopes. A TimeStampedData envelope, described in [RFC5544], binds a file with one or more time-stamp tokens obtained for that file. A media type registration, lacking in [RFC5544], enhances shareability.

このドキュメントでは、TimeStampedDataエンベロープの新しいメディアタイプを定義しています。[RFC5544]に記載されているタイムスタンペドダタエンベロープは、そのファイルに対して取得された1つ以上のタイムスタンプトークンを含むファイルをバインドします。[RFC5544]がないメディアタイプの登録は、共有可能性を高めます。

Applications which use this media type:
